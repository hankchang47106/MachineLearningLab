# Lux AI Challenge Session 2
Refer: https://github.com/Lux-AI-Challenge/Lux-Design-S2

## Create Enviroment
- list conda venv
```shell
$ conda env list 
```
  
- create a venv in python
```shell
$ conda create --name lux_ai_s2 python=3.7 
```

- Activate the env
```shell
$ conda activate lux_ai_s2 
```

## Install Package
```shell
$ pip install --upgrade luxai_s2

```
